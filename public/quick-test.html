🧪 Quick Test Page

This page will test if your tools extension files are working.

📋 Instructions

  1. Upload this file (quick-test.html) to your /public/ folder
  2. Upload tools-extension.js to your /public/ folder
  3. Open this page in your browser
  4. Click the test buttons below

🐛 Debugging Tips

If tests fail:

  1. Open browser console (F12)
  2. Look for error messages in red
  3. Common errors: 
    Failed to load: tools-extension.js → File not found
Unexpected token → File corrupted
404 Not Found → Wrong location

📁 File Locations

Both files must be in the same folder:

/public/
├── quick-test.html    ← This test page
├── tools-extension.js ← Extension code
└── ...

After tests pass:

/public/
├── tools-extension.js ← Upload this
├── index.html         ← Upload updated version
└── ...
